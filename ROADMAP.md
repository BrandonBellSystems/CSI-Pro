# CSI Pro Roadmap

> Public development direction

## Near-Term
- [ ] Cloud Sync (optional subscription layer)
- [ ] Historical score tracking with trend visualization
- [ ] Multi-account portfolio comparison view

## Medium-Term
- [ ] Mobile app wrapper (Capacitor/Tauri)
- [ ] Slack/Teams integration
- [ ] Advanced onboarding sequence automation

## Long-Term
- [ ] Team collaboration features
- [ ] CRM integration (read-only, local processing)
- [ ] Custom scoring model configuration

---

*Cloud Sync will be an optional subscription. The core tool will always remain a one-time purchase. Early buyers are grandfathered into all Pro features they paid for.*
